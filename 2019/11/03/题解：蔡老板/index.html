题解：蔡老板

有 $n$ 个物品，每个物品价值为 $2^{a_i}$ ，重量为 $w_i$。

选择一些物品，求使得重量 $\ge k$ 的最小价值。

$n\le10^5,a_i\le30,w_i\le10^9,k\le\sum w_i$

solution

考虑二分答案 $k$ ，分三类讨论。

如果 $k$ 为偶数，将 $a_i=0$ 的物品按价值从大到小分组，$n$ 个 $a_i=0$ 的物品可以转换为 $\left \lfloor \frac n2 \right \rfloor$ 个 $a_i=1$ 的物品。

如果 $k$ 为奇数，取走 $w_i$ 最大的物品，再按 $k$ 为偶数时分组。

如果 $a_i=0$ 时，将 $a_i,k$ 变为 $\frac{a_i}2,\frac k2$ 即可。

维护一个堆即可

时间复杂度：$\Theta(n\log{\sum 2^{a_i}}\log n)$
